# moveto
Extract RTP listing data from moveto.com.

  Default: Maximum price: 500,000, Zip: 28685, Beds: 3, Baths: 2 

  Change the default as shown [here](https://github.com/xgirma/moveto/blob/master/HOWTO.md)
